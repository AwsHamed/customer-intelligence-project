# 🎯 Customer Intelligence Analysis

## 📊 Project Overview
Customer segmentation and marketing response prediction using Machine Learning.

## 🎯 Key Results
- Customer segmentation into **2 segments**: Elite Spenders & Frugal Youngsters
- Best model: **AdaBoost with 89.06% accuracy**
- Dimensionality reduction using PCA (preserved 46.5% variance)

## 🛠️ Technologies Used
- Python, pandas, numpy
- scikit-learn (PCA, K-Means, AdaBoost)
- XGBoost, LightGBM
- matplotlib, seaborn

## 📈 Model Performance
| Model | Accuracy |
|-------|----------|
| AdaBoost | **89.06%** |
| LightGBM | 87.28% |
| XGBoost | 87.05% |

## 🚀 How to Run
1. Clone this repository
2. Install requirements: `pip install -r requirements.txt`
3. Open `Customer_Intelligence.ipynb` in Jupyter Notebook
